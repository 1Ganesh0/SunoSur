#  SunoSur – Music Streaming App

SunoSur is a web-based music streaming application that allows users to search and play songs from YouTube with a modern UI inspired by Apple Music.

##  Live Demo
[SunoSur](https://mpo5twprpnchouwwu7r5x4.streamlit.app/)

##  Features
-  Search songs using YouTube API
-  Play music instantly
-  Queue system
-  Playlist system
-  Modern UI (Apple Music inspired)
-  Works on mobile (PWA supported)

## Tech Stack
- Python
- Streamlit
- YouTube Data API
- Requests

##  How to Run

1. Clone the repo
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   streamlit run app.py

(OR)

   Click the link provided in Live Demo


##  Deployment

Deployed using Streamlit Community Cloud.
